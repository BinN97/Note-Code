Linux下在一个文件名/目录名前加上一个.表示这个为隐藏
Linux下每个目录下都有两个特殊的路径
.:当前目录
..:上一级目录
/:根目录
~：主目录

ls:查看当前目录下的字文件或者字目录
ls 路径:查看这个路径下的子文件或者子目录
ls -a:查看当前目录下包含隐藏文件在内的所有文件和目录
ls -l:查看目录下的子文件或者子目录的详细信息
ls -l 路径:查看路径下的子文件或者子目录的详细信息
ls -l 文件：单独查看文件的详细信息
ls-al：查看隐藏文件在内的所有子文件或者子目录的详细信息

时间为最后一次修改时间
所有目录的大小都为4096字节

r--4      rwx  rwx  r-x     rw-rw-r--
w--2      421  421  401     6  6  4(八进制)
x--1      7    7    5
---0   
新创建的文件权限为664
新创建的目录权限为775
clear：清屏
chmod：修改文件权限
   chmod 要修改的权限值 要修改的文件名
   chmod 774 qw.sh
cd：切换当前所在目录
相对路径：相对某个目录的路径
绝对路径：从根目录下开始的路径
  cd /    切换到根目录
  cd ~    切换到主目录
  cd .    切换到当前路径
  cd ..   切换到上一级目录
  cd 回车 切换到主目录
cd ~/Desktop
cd ../Niub/day03
touch：如果文件存在则更新最后一次修改时间，不存在则创建一个新的空文本文件
   touch 文件名
mkdir：创建一个新的空目录 如果存在则创建失败，不存在则创建成功
     mkdir 目录
mkdir dir
mkdir d1 d2 d3
mkdir -p dir/dir..-p创建不同级目录
rmdir:删除空目录                    鸟哥的Linux私房菜 UNix环境编程
   rmdir 路径/目录
rm：删除文件或者删除目录
     rm 文件名
     rm -f 文件名
     rm -r 目录名
     rm -rf 目录名
cp：复制文件或目录
   cp 源文件 目标文件
   cp -r 源目录 目标目录
注意:查看目标文件或者目录有没有同名的再复制/剪切
mv: 剪切文件或者目录
   mv 源文件/目录 目标文件/目录
  重命名的作用
man：shell命令或者C语言的帮助手册
   man [章节号] 命令/函数名
ping：查看两台电脑是否能连接
   ping ip/域名
   tianqi.sogou.com
   www.iciba.com
ifconfig:获取自己电脑的网络信息
   ifconfig
sudo：临时获取权限
